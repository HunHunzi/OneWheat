# JavaScript 补全

# var let const

  1. var 无块级作用域，有函数作用域（慎用），它会在所有函数的顶部声明变量，但是不会赋值，所以会出现变量提升的现象，即使在声明之前使用变量也不会报错，但是值为 undefined

  2. let 有块级作用域，没有函数作用域，它会在块级作用域的顶部声明变量，但是不会赋值，所以会出现变量提升的现象，但是在声明之前使用变量会报错

  3. const 无块级作用域，没有函数作用域，它会在块级作用域的顶部声明变量，但是不会赋值，所以会出现变量提升的现象，但是在声明之前使用变量会报错，它声明的变量必须赋值，且不能修改

  4. var let const 声明的变量都会挂载到 window 上，但是 var 声明的变量可以被覆盖，let 和 const 声明的变量不能被覆盖

  5. 对于一些常量，对象，函数都可以用 const 赋值

# 展开运算符

  1. 展开运算符可以展开数组，对象，字符串，类数组对象，它会将对象的属性展开，将数组的元素展开，将字符串的字符展开，将类数组对象的元素展开(...arr, ...obj, ...str, ...arguments) 展开相当于浅拷贝

# 箭头函数

  1. 箭头函数的定义：() => {};
  2. 它没有 this，没有 arguments，没有 prototype，不能作为构造函数，不能使用 new，不能使用 call，apply，bind... （arguments 可以用...args 代替,它用来保存函数的参数）

const fn =function(a){ return "hello"; };

相当于： const fn = a => "hello"; //如果只有一个参数，可以省略括号

()=>{} 相当于 function(){},没有参数的写法 (a,b,c)=>{} 相当于 function(a,b,c){},有参数的写法

  1. 返回值必须是表达式，如果是对象，必须用()包裹
  2. 箭头函数的 this 指向它的上一级作用域的 this，如果上一级作用域没有 this，就指向 window； 例如： const obj = { fn: function(){ console.log(this); //this => obj const fn2 = () => { console.log(this); //this => obj } fn2(); } }
  3. 箭头函数的 this 指向它的上一级作用域的 this，如果上一级作用域没有 this，就指向 window；
  4. 箭头函数的 this 无法通过 call，apply，bind 改变
  5. 箭头函数无法作为构造函数，不能使用 new

# 模块化

  1. 默认导出：export default xxx; //导出的是一个值，可以是任意类型
  2. 命名导出：export const xxx = xxx; //导出的是一个变量，可以是任意类型

# 类

  1. 类的定义：class xxx {}
  2. 类的继承：class xxx extends xxx {} //继承父类的属性和方法，当在子类中重写父类的构造函数时，必须 super()，调用父类的构造函数
  3. 类对象的创建：new xxx() //创建类的实例
  4. 类的构造函数：constructor() {}
  5. 类中的所有代码都会在严格模式下执行，其中的 this 指向类的实例，而不是 window
  6. 对类中 this 的绑定，this.xxx = this.xxx.bind(this); //在构造函数中绑定 this,使其指向类的实例
  7. 在类中尽量使用箭头函数，因为箭头函数没有 this，它的 this 指向它的上一级作用域的 this，如果上一级作用域没有 this，就指向 window；

# 数组方法补全

  1. map() //返回一个新数组，数组中的每个元素都会执行回调函数，回调函数的返回值作为新数组的元素 例： arr.map((item, index) => {return item + 1;}) //每个元素都加 1
  2. filter() //过滤，返回符合条件的值的新数组
  3. forEach() //遍历数组，没有返回值
  4. some() //判断数组中是否有符合条件的元素，有一个符合条件就返回 true，否则返回 false
  5. find() //查找数组中符合条件的元素，返回第一个符合条件的元素，没有符合条件的元素返回 undefined
  6. reduce() //累加器，返回一个值，它会遍历数组，将数组中的元素依次传入回调函数，回调函数的返回值作为下一次调用回调函数的第一个参数，第一次调用回调函数时，第一个参数为初始值，如果没有初始值，第一次调用回调函数时，第一个参数为数组的第一个元素，第二个参数为数组的第二个元素，第二次调用回调函数时，第一个参数为第一次调用回调函数的返回值，第二个参数为数组的第三个元素，以此类推，直到遍历完数组，返回最后一次调用回调函数的返回值
